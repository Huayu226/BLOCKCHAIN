# 🧠 Enhanced DAO DApp

這是一個基於 React + Vite 所開發的去中心化應用程式（DApp），可連接至 EnhancedDAO 智能合約，提供創建提案、投票、執行提案等功能。

---

## 🚀 專案啟動方式

### 1️⃣ 安裝相依套件

請先在專案資料夾內執行以下指令來安裝需要的依賴：

npm install
### 2️⃣ 啟動開發伺服器

npm run dev
打開瀏覽器前往 http://localhost:5173 即可進入 DApp 頁面。
